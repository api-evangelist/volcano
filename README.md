# Volcano (volcano)
Volcano is a CNCF incubating batch processing and HPC scheduler for Kubernetes. It provides gang scheduling, fair-share scheduling, queue management, and integrates with ML and big data frameworks.

### Volcano Batch Scheduling API
Kubernetes CRDs for batch workload management with Job and Queue resources supporting gang scheduling, fair sharing, and integration with TensorFlow, PyTorch, Spark, and MPI.
